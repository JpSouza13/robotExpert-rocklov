# robotExpert-rocklov
Curso Robot Expert.

Automaçao de testes de API e da Aplicação web da Rocklov, utilizando o RobotFramework.
* Tecnologias e ferramentas utilizadas :
- RobotFramework (Python)
- Insomnia
- Swagger
- Playwright(Python)
- MongoDB
- Heroku

https://rocklov-jp-web.herokuapp.com/
